- Aimless progressions
- Voice leading issues
- Chromatic problems
- Cadential weaknesses

### MELODY (15 diagnostics)
- Contour: direction, stepwise, angular, predictable
- Leaps: large, tritones, consecutive
- Range: narrow, wide, placement
- Rhythm: repetition, motives, phrasing
- Harmony fit: avoiding chord tones, too many NCTs

### TEXTURE (13 diagnostics)
- Density thick: voices, spacing, hierarchy
- Density thin: voices, gaps, bass
- Spacing: close, crossing, unbalanced
- Rhythm: homorhythm, saturation
- Static: same voices, no change

### RHYTHM (14 diagnostics)
- Momentum: long notes, no acceleration, equal durations
- Pacing: too busy, too fast, too slow
- Predictable: mechanical, regular phrases, no syncopation
- Meter: unclear downbeats, wrong signature, competing layers
- Phrasing: obscured boundaries, no agogic accent

### FORM (14 diagnostics)
- Proportion: opening long, development short, rushed ending
- Transitions: abrupt, overlap, key changes
- Boundaries: weak cadences, phrase lengths, no markers
- Predictable: conventional, no surprises
- Return: too literal, weak arrival, no completion

### MULTIPLE/UNSURE (4 diagnostics)
- Interconnected issues
- Intuitive feelings

- Verbal description pathway

## Usage Instructions

### Installation:
1. Save `CompositionTutor.qml` to `~/Documents/MuseScore4/Plugins/`
2. Restart MuseScore
3. Enable in Plugins → Plugin Manager

### Using the Plugin:
1. Select measures in your score
2. Run: Plugins → Composition Tutor
3. Choose category (Harmony, Melody, Texture, Rhythm, Form, Multiple/Unsure)
4. Navigate through 2-3 levels of questions
5. Receive diagnostic with:
   - Problem reframing
   - 3-4 solution spaces
   - OpenMusicTheory link
   - Canonical text recommendations
   - "Discuss with Claude AI" button

## Design Philosophy Maintained Throughout

The plugin embodies a **Socratic, non-prescriptive pedagogical approach:**

1. **Questions before answers** - Diagnostic tree helps users articulate the problem
2. **Multiple valid solutions** - No single "correct" answer
3. **Composer agency** - User decides which approach fits their goals
4. **Theory as tool** - Vocabulary enables thinking, not rules to follow
5. **Conditional framing** - "If your goal is..." respects compositional intent
6. **Solution spaces** - Categorical strategies, not note-level prescriptions
7. **Canonical grounding** - Connects to established theoretical literature
