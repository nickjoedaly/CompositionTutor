## Complete Diagnostic List

1. **diag_texture_thick_voices** - Too many simultaneous voices
   - OMT: Texture
   - Solutions: Selective reduction, voice pairing, staggered entrances, textural relief

   - OMT: Chords in SATB Style

3. **diag_texture_thick_hierarchy** - No registral hierarchy

### Density Issues - Too Thin (3)
4. **diag_texture_thin_voices** - Not enough voices for fullness
   - OMT: Texture
   - Solutions: Add voices, octave doublings, wider spacing, pedal points

5. **diag_texture_thin_gap** - Too much registral gap
   - OMT: Chords in SATB Style
   - Solutions: Close position, inner voice filling, register consolidation, pedal/drone

6. **diag_texture_thin_bass** - Weak bass foundation
   - OMT: Chords in SATB Style
   - Solutions: Bass motion, arpeggiation, octave reinforcement, rhythmic activation

### Spacing Issues (3)
7. **diag_texture_spacing_close** - Voices too close together
   - OMT: Chords in SATB Style
   - Solutions: Open position, register separation, selective voicing, octave displacement

8. **diag_texture_spacing_crossing** - Voices cross frequently
   - OMT: Chords in SATB Style
   - Solutions: Maintain registral hierarchy, voice identity, avoid crossing, brief crossings only

9. **diag_texture_spacing_unbalanced** - Unbalanced register distribution
   - OMT: Chords in SATB Style

    - OMT: Texture
    - Solutions: Selective rest, voice reduction, rhythmic contrast, breathing space

### Static Texture Issues (2)

## Testing Path Examples

**Example 1: Muddy thick texture**
Start → Texture → Too dense or thick → Narrow spacing creates muddiness
→ **DIAGNOSTIC:** Registral crowding
- Solutions: Open position, register separation, octave doublings

**Example 2: Hollow thin texture**
Start → Texture → Too sparse or thin → Too much registral gap
→ **DIAGNOSTIC:** Empty middle register
- Solutions: Close position, inner voice filling, register consolidation

**Example 3: Voices crossing**
Start → Texture → Voices overlap or clash → Voices cross frequently
→ **DIAGNOSTIC:** Voice independence loss
- Solutions: Maintain registral hierarchy, voice identity, avoid crossing

**Example 4: No textural change**
Start → Texture → Texture doesn't change → Same number of voices throughout
→ **DIAGNOSTIC:** Textural monotony
- Solutions: Voice addition/subtraction, textural variation, buildup/reduction
